\section{Commands configuration}
User can configure bot commands in file cmd.yaml.
\begin{lstlisting}[language=yaml]
"command_name":
    part: yes,
    hash: "{@year@}",
    icase: yes,
    word: "regex" / [word1, word2, word3, ...]
    command: command1 / [command1, command2, command3, ...]
\end{lstlisting}
